CONFIGURATION
-------------

### Database

Change the file `config/db.php.example` to `db.php`.
 Edit the file `db.php` with real data, for example:

```php
return [
    'class' => 'yii\db\Connection',
    'dsn' => 'mysql:host=localhost;dbname=yii2basic',
    'username' => 'root',
    'password' => '1234',
    'charset' => 'utf8',
];
```
